About Web Testing Service

The Web Testing Service is a web-based runner indicates how well the Crosswalk Project enable your product supports the HTML5 standard and related specifications on cross platforms. Measure your product with the latest web technology and comprehensive standard web test suites always, and compare the results with other web products you care about.

The Web Testing Service does not only host the latest HTML5 standard tests, but also involves the tests for the hottest web technologies, e.g. Real Sense, SIMD, Presentation APIs, as well as supports users to run them through browsers or web runtimes. Crosswalk Project QA team have been devoted on Web Testing Service, provided customization test plan according to user needs, and hoping that by using Web Testing Service, users could get an idea of improving their product and the web as a development platform.

The set of tests run is configurable via the specification name selection or searching. Especially, the searching input provides candidates which bases on specification name or category name:

  • css3 - add all tests of "css3" specification category to the execution list.
  • SIMD - add all tests of "SIMD" specification to the execution list.
